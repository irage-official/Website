# وب‌سایت ایراژ

وب‌سایت رسمی پروژه ایراژ (تقویم مستقل ایرانی)

## درباره

این وب‌سایت برای معرفی و ارائه اطلاعات درباره اپلیکیشن ایراژ طراحی شده است. وب‌سایت شامل صفحات زیر است:

- **صفحه اصلی**: معرفی پروژه و ویژگی‌ها
- **درباره ما**: اطلاعات کامل درباره پروژه و مأموریت آن
- **منابع**: لیست منابع و مراجع استفاده شده
- **شرایط استفاده**: شرایط و قوانین استفاده از خدمات
- **پالیسی**: پالیسی حریم خصوصی

## ویژگی‌ها

- طراحی ریسپانسیو و مدرن
- استفاده از تم رنگی و فونت‌های پروژه کلندر
- مدال حمایت (Donation Modal)
- ناوبری ساده و کاربرپسند
- پشتیبانی کامل از زبان فارسی (RTL)

## ساختار پروژه

```
Calendar website/
├── assets/
│   ├── css/
│   │   └── style.css          # استایل‌های اصلی
│   ├── js/
│   │   └── main.js            # اسکریپت‌های جاوااسکریپت
│   ├── fonts/
│   │   ├── YekanBakh-VF.ttf  # فونت فارسی
│   │   └── Inter.ttf          # فونت انگلیسی
│   └── images/
│       ├── logo_centered.png  # لوگو
│       └── logo_launcher.svg  # لوگو SVG
├── pages/
│   ├── about.html             # درباره ما
│   ├── terms.html             # شرایط استفاده
│   ├── policy.html            # پالیسی حریم خصوصی
│   └── resources.html         # منابع
├── index.html                 # صفحه اصلی
└── README.md                  # این فایل
```

## تم رنگی

وب‌سایت از همان تم رنگی پروژه کلندر استفاده می‌کند:

- **Primary**: #735BF2 (بنفش)
- **Secondary**: #DE496E (قرمز/صورتی)
- **Background**: #F1F5F9 (خاکستری روشن)
- **Text**: #1E293B (خاکستری تیره)

## فونت‌ها

- **YekanBakh**: برای متن‌های فارسی
- **Inter**: برای متن‌های انگلیسی

## استفاده

برای مشاهده وب‌سایت، کافی است فایل `index.html` را در مرورگر باز کنید.

### استفاده محلی

```bash
# با استفاده از Python
python3 -m http.server 8000

# یا با استفاده از Node.js (http-server)
npx http-server
```

سپس به آدرس `http://localhost:8000` مراجعه کنید.

## توسعه

برای توسعه وب‌سایت:

1. فایل‌های HTML را در پوشه ریشه و `pages/` ویرایش کنید
2. استایل‌ها را در `assets/css/style.css` تغییر دهید
3. اسکریپت‌ها را در `assets/js/main.js` ویرایش کنید
